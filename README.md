# Sales Advertising Analysis with Linear Regression

This project analyzes the relationship between advertising spending and sales using multiple linear regression. The model predicts sales based on advertising budgets allocated to TV, Radio, and Newspaper channels.

## 📋 Dataset

The dataset contains 100 observations with the following features:
- **TV**: Advertising budget spent on TV (in thousands of dollars)
- **Radio**: Advertising budget spent on Radio (in thousands of dollars)
- **Newspaper**: Advertising budget spent on Newspaper (in thousands of dollars)
- **Sales**: Sales of the product (in thousands of units)

## 🛠️ Technologies Used

- Python 3.12
- pandas - Data manipulation and analysis
- numpy - Numerical computations
- scikit-learn - Machine learning models and evaluation
- matplotlib - Data visualization

## 📊 Model Performance

The multiple linear regression model was evaluated using:

- **Mean Absolute Error (MAE)**: 1.36
- **Root Mean Square Error (RMSE)**: 2.10

These metrics indicate that the model's predictions are, on average, within approximately 1.36 thousand units of actual sales.
